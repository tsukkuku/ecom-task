## Установка

1. Скачать исходный код
```bash
git clone https://github.com/tsukkuku/ecom-task
cd ecom-task
```

2. Установите зависимости
```bash
npm install
```

## Запуск

1. Запуск на локальном сервере
```bash
npm run dev
```

2. Сборка
```bash
npm run build
```

**Собранные файлы будут в папке dist/**

3. Просмотр собранного проекта
```bash
npm run preview
```
